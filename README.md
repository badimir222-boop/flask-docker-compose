# 🐳 Production-Ready Docker Compose Project

# О проекте
Веб-приложение на Flask с PostgreSQL и Redis, оркестрируемое Docker Compose.  
Настроены healthchecks, автоматические тесты CI/CD через GitHub Actions.

# Стек технологий
- *Flask** (Python)
- *PostgreSQL 15**
- *Redis**
- *Docker / Docker Compose**
- *GitHub Actions** (CI/CD)

 Запуск
```bash
docker compose up -d
